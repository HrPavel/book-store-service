BOOK-STORE-SERVICE
======

This project implements a simple bookstore, that allows you to create / update books and view all existing books.

## Technologies
 - Java 8
 - Spring Boot 2
 - H2 database
 - Liquibase
 - Junit 5
 
## Run the application
Tests
```
 ./mvnw clean test
```
Application
```
 ./mvnw spring-boot:run
```
